# テンプレートマッチング（パラメータキャリブレーション）PoC 結果レポート

## 実施内容
ユーザーが指定した「正解の枠」を分析し、その特徴（サイズ、色、面積）に基づいて画像全体から類似した枠を検出するスクリプト `poc/template_matching_poc.py` を作成・実行しました。

## テスト条件
- **対象画像**: `パンターG型_001.jpg`
- **テンプレート（ユーザー指定想定）**:
  - 座標: `(100, 747, 932, 320)` （以前の検出結果から推定）
  - 分析結果:
    - サイズ: 932 x 320 px
    - アスペクト比: 2.91
    - 主要色: 黒 (Black)

## 検出結果
- **検出数**: 4個
- **内容**: テンプレートと類似したサイズ・色の枠が検出されました。

## 評価
このアプローチ（One-Shot Learning / Parameter Calibration）は正常に機能しています。
ユーザーが指定した枠の特徴（「このくらいの大きさ」「この色」）を即座に学習し、フィルタリング条件として適用できています。

## メリット
1. **調整不要**: 画像ごとに枠のサイズや線種が違っても、ユーザーが1つ指定すれば自動で追従します。
2. **直感的**: 「これと同じものを探して」という指示はユーザーにとって分かりやすいです。

## 次のステップ
このロジックを `apps/admin-tool` に統合することを推奨します。
